# FTE_Driver
FTE控制器、架構、通訊、接線、範例程式、操作說明
首先介紹主控制器，它經由USB轉RS232的線，
將電腦的串列資料(Serail data)傳入主控器。

主控器在將傳入的資料轉成CANBus格式，發送給驅動器運動指令。
